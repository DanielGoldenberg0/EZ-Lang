# ezLang
ezLang is a programming language with an extremely simple syntax designed for fast prototyping

# Syntax

## Printing to the Console
  ```python
  output "Hello, world!"
  ```
Output: `"Hello, world!"`

## Math Operations
  ```python
  add 5 3
  ```
Output: `8.0`
