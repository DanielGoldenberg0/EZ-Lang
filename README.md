# ezLang
ezLang is a programming language with an extremely simple syntax designed for fast prototyping

# Syntax

## Printing to the Console
  ```python
  output "Hello, world!"
  ```
Output: `"Hello, world!"`

## Math Operations
  ```python
  operate 5 - 4
  ```
Output: `1.0`
  ```python
  operate 3 ** 2
  ```
Output: `8.0`

## If Statements
  ```python
  if 3 == 3
  ```
Output: `True`
